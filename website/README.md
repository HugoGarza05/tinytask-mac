# TinyTaskMac Website

Marketing site and download surface for TinyTaskMac.

## Development

```bash
cd website
npm install
npm run dev
```

## Build

```bash
cd website
npm run build
```

The site reads release data from [`public/release-manifest.json`](/Users/hugogarza/Projects/gpt-5.4-projects/tinytask-mac/website/public/release-manifest.json). Update that file before publishing a new release or use the release helper scripts in [`scripts/`](/Users/hugogarza/Projects/gpt-5.4-projects/tinytask-mac/scripts).
